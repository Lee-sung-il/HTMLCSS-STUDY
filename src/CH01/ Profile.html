자기 소개서 페이지

이성일

안녕하세요 저는 웹개발자 지망생 이성입니다. 다양한 웹 기술을 다루면 사용자 친화적인 웹 애플리케이션을 만든는것을 좋아합니다.

취미

저는 유튜브를 보는것을 좋아하면, 새로 나오는 기술이라던지 찾아보는것 좋아합니다.

좋아하는 음식

  • 초밥
  • 돈까스
  • 덮밥
  • 회

전공

대학에서 소프트웨어 컨테츠계열을 전공 하였으면 다양한 프로그래밍 언어와 알고리즘을 공부 하였습니다. 이 과정에서 소프트웨어 개발에 대한 깊은 이해를 쌓았습니다.

프론트엔드 개발을 시작하게 된 이유

예전에 웹 개발을 하면서 화면을 구현하는 작업을 즐겼던 기억이 있습니다. 비록 지금은 많이 잊었지만, 그때의 재미를 다시 느끼고 싶어 다시 공부를 시작했습니다. 결국, 사용자 경험을 만족시키는 프런트엔드 개발자가 되고 싶다는 목표를 가지게 되었습니다.
